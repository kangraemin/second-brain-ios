# UX Guide

---

## 핵심 원칙

> **저장은 1탭, 정리는 앱이, 검색은 AI가.**

---

## 1. 저장 (Share Extension)

```
다른 앱에서 공유 → Stash 아이콘 탭 → 저장 완료
모달 없음. 토스트: "저장 완료 ✓" (선택적 "컬렉션 변경" 링크)
```

- Share Extension에서는 URL만 저장.
- 메타데이터 추출, 임베딩은 메인 앱에서 백그라운드 처리.

---

## 2. 메인 화면

```
검색바
[전체] [영상] [장소] [쇼핑] [아티클] [인스타]  ← 자동 분류 필터 칩
2열 카드 그리드 (최신순)
```

필터 칩 매핑:
- 영상: YouTube, Instagram Reel
- 장소: 네이버지도, 구글맵
- 쇼핑: 쿠팡, 네이버쇼핑
- 아티클: 블로그, 뉴스, 일반 웹
- 인스타: Instagram 포스트

---

## 3. 소스별 카드

| 소스 | 카드 내용 |
|------|---------|
| YouTube | 썸네일(16:9) + 제목 + 채널명 + 재생시간 |
| Instagram | 이미지(1:1) + @username + 캡션 |
| 장소 | 장소 이미지 + 장소명 + 주소 + 지도앱 버튼 |
| 쇼핑 | 상품 이미지 + 상품명 + 가격 |
| 웹 | OG 이미지 + 제목 + 도메인 |

---

## 4. 검색

1. 키워드 검색 (`localizedStandardContains`)
2. 시맨틱 검색 (`NLContextualEmbedding` + 코사인 유사도)
3. 결과 병합 랭킹

---

## 5. 점진적 정리

| 시점 | 동작 |
|------|------|
| Day 1 | 콘텐츠 타입 자동 분류 (유저 노력 0) |
| 일정량 누적 | 앱이 컬렉션 생성 제안 |
| 장기 사용 | AI 검색으로 정리 없이도 검색 가능 |

수동 컬렉션은 파워 유저용 옵션.
